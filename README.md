# Pak Koding 🚀
**Platform Edukasi Pemrograman & Kecerdasan Artifisial Modern**

Pak Koding adalah platform pembelajaran berbasis web yang dirancang khusus untuk pemula yang ingin terjun ke dunia teknologi dengan cara yang santai, terstruktur, dan mudah dipahami.

---

## ✨ Fitur Utama
- 📚 **Kurikulum Terstruktur**: Materi belajar mulai dari dasar web (HTML, CSS, JS) hingga konsep modern Kecerdasan Artifisial (AI).
- 💻 **Live Code Preview**: Lihat hasil kodingmu secara langsung di browser tanpa perlu setup yang rumit.
- 🤖 **Modul AI Terpadu**: Belajar sejarah, cabang, hingga etika AI dalam satu jalur pembelajaran.
- 📈 **Progress Tracking**: Simpan kemajuan belajarmu secara otomatis di browser (Local Storage).
- 📱 **Desain Responsif & Premium**: Pengalaman belajar yang nyaman di perangkat apa pun, mulai dari smartphone hingga desktop.
- 🌑 **Modern UI/UX**: Antarmuka bersih dengan sentuhan estetika premium (Glassmorphism & Micro-animations).

---

## 🛠️ Tech Stack
Proyek ini dibangun menggunakan teknologi mutakhir untuk performa maksimal:
- **Framework**: [Astro v6](https://astro.build/) (Static Site Generation & Hybrid Rendering)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Adapter**: [@astrojs/node](https://docs.astro.build/en/guides/integrations-guide/node/)
- **Language**: TypeScript & Vanilla JavaScript
- **Deployment Ready**: Optimized for Cloudflare Pages / Vercel / Netlify

---

## 🚀 Cara Menjalankan Secara Lokal

### Prasyarat
- Node.js (Versi 18 atau lebih baru)
- NPM atau Bun

### Langkah Instalasi
1. **Clone repositori**:
   ```bash
   git clone <url-repositori-anda>
   cd pak-koding
   ```

2. **Instal dependensi**:
   ```bash
   npm install
   ```

3. **Jalankan server pengembangan**:
   ```bash
   npm run dev
   ```
   Buka `http://localhost:4321` di browsermu.

---

## 📖 Struktur Proyek
- `/src/pages`: Halaman utama dan rute dinamis materi.
- `/src/components`: Komponen UI yang dapat digunakan kembali (Logo, Header, Sidebar, dll).
- `/src/data`: Pusat data materi (`lessons.ts`) dan FAQ.
- `/public`: Aset statis seperti ikon, gambar, dan favicon.
