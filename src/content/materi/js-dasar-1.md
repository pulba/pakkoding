      JavaScript membuat website hidup dan bisa melakukan aksi pintar.
              
      - **Tag <script>**: Tempat kita menaruh semua perintah koding JavaScript
      di file HTML.

      - **alert()**: Perintah untuk memunculkan **Pesan Pop-up** (kotak
      peringatan) di layar.

      - **console.log()**: Perintah untuk memunculkan tulisan di bagian
      "Inspect" (untuk mengecek apakah kodinganmu benar).
    code: |-
      <script>
          alert("Selamat Datang!");
      </script>
