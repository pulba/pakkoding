---
title: "Jarak: Margin & Padding"
category: CSS
order: 24
description: Memberi ruang agar elemen tidak berdempetan.
content:
  - explanation: >-
      Konsep "Kotak" di CSS sangat penting untuk mengatur jarak agar web tidak
      terlihat berantakan.
              
      - **Padding**: Jarak **di dalam** kotak (antara teks dan garis tepi
      kotak).

      - **Margin**: Jarak **di luar** kotak (jarak antar satu kotak dengan kotak
      lainnya).

      - **Border**: Garis tepi kotak. Bisa diatur ketebalan (px) dan warnanya.
    code: |-
      div { 
          border: 2px solid black;
          padding: 20px; 
          margin: 30px; 
      }
    challenge: Manakah yang memberikan jarak antara kotak satu dengan kotak lainnya
      di luarnya, Margin atau Padding?
