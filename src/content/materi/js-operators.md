description: Menghitung angka dan membandingkan kebenaran.
content:
  - explanation: >-
      JavaScript sangat jago matematika dan perbandingan.
              
      - **Matematika Dasar**: Tambah (+), Kurang (-), Kali (*), dan Bagi (/).

      - **Sisa Bagi (%)**: Disebut Modulo, berguna untuk mengecek bilangan
      genap/ganjil.

      - **Perbandingan**: Digunakan untuk mengecek apakah dua nilai sama.
          - === (Sama Persis): Mengecek nilai DAN tipe datanya (SANGAT disarankan).
          - !== (Tidak Sama).
    code: |-
      let a = 10;
      let b = 5;

      console.log(a + b);    // 15
      console.log(a === 10); // true (benar)
      console.log(b > 10);   // false (salah)
    challenge: Operator apakah yang paling disarankan di JavaScript untuk mengecek
      apakah dua nilai itu "Sama Persis" nilainya maupun tipe datanya?
