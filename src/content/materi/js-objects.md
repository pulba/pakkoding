---
title: Objek (Objects)
category: JS
order: 43
description: Menyimpan data kompleks yang memiliki atribut.
content:
  - explanation: >-
      Jika Array seperti daftar belanja, maka Objek seperti KTP seseorang. Objek
      menyimpan data yang memiliki penjelasan (properti).
              
      - **Kurung Kurawal {}**: Objek dibungkus menggunakan kurung kurawal.

      - **Key-Value Pair**: Data disimpan berpasangan, ada nama properti (key)
      dan nilainya (value).

      - **Titik (.)**: Cara paling mudah untuk memanggil nilai di dalam objek
      adalah menggunakan titik.
    code: |-
      let siswa = {
          nama: "Budi",
          kelas: "10A",
          umur: 15
      };

      // Mengambil nama siswa
      console.log(siswa.nama);

      sebuah properti dari dalam objek JavaScript?
