title: Perulangan (Loops)
category: JS
order: 46
description: Mengerjakan tugas ribuan kali dalam sedetik.
content:
  - explanation: >-
      Komputer tidak pernah bosan. Jika kamu butuh mencetak tulisan 100 kali,
      jangan ditulis manual, gunakan Loop!
              
      - **for**: Loop yang paling sering dipakai. Terdiri dari 3 bagian: titik
      awal, batas akhir, dan penambahan.

      - **i++**: Singkatan dari "i = i + 1" (tambahkan 1 di setiap putaran).

      - **while**: Berulang terus SELAMA kondisinya masih bernilai benar (true).
    code: |-
      // Mencetak angka 1 sampai 5
      for (let i = 1; i <= 5; i++) {
          console.log("Perulangan ke-" + i);
      }
    challenge: Perintah "i++" di dalam sebuah loop for memiliki fungsi yang sama
      dengan apa?
