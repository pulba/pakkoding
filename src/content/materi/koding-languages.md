content:
  - explanation: >-
      Sama seperti manusia yang berbicara dalam berbagai bahasa (Indonesia,
      Inggris, Jepang), komputer juga punya banyak bahasa pemrograman. Setiap
      bahasa punya kelebihan masing-masing.
              
      - **Python**: Bahasa yang paling mudah dipelajari pemula, sangat populer
      untuk AI dan Data Science.

      - **JavaScript**: Raja-nya website. Digunakan untuk membuat website
      interaktif (frontend & backend).

      - **Java**: Banyak dipakai untuk aplikasi Android dan sistem perusahaan
      besar.

      - **C / C++**: Bahasa yang sangat cepat, digunakan untuk membuat game,
      sistem operasi, dan perangkat keras.

      - **PHP**: Bahasa khusus untuk server website (backend), dipakai oleh
      WordPress dan Facebook versi awal.
    code: |-
      // Bahasa JavaScript
      console.log("Halo dari JavaScript!");

      // Bahasa Python (contoh saja, tidak jalan di sini)
      // print("Halo dari Python!")

      // Bahasa Java (contoh saja)
      // System.out.println("Halo dari Java!");
